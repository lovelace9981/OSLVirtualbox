## Requisitos mínimos:

- __CPU:__ 4 núcleos físicos o 2 núcleso físicos con SMT
- __RAM:__ 8GiB de RAM mínimo (Windows y GNU/Linux). 
- __Tecnologías de virtualización__ 
  - __VT-X__ - Sistemas Intel
  - __AMD-V__ - Sistemas AMD
